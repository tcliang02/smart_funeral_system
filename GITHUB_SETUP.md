# GitHub Repository Setup

## Step 1: Create Repository on GitHub

1. Go to https://github.com/new
2. Repository name: `smart_funeral_system`
3. Description: "Smart Funeral System - Digital funeral service management platform"
4. Choose **Private** or **Public** (your choice)
5. **DO NOT** check:
   - ❌ Add a README file
   - ❌ Add .gitignore
   - ❌ Choose a license
6. Click **"Create repository"**

## Step 2: Push Your Code

After creating the repository, run these commands:

```bash
git push -u origin main
```

If you get authentication errors, you may need to:
- Use a Personal Access Token instead of password
- Or use SSH: `git remote set-url origin git@github.com:tcliang2002/smart_funeral_system.git`

## Step 3: Verify Push

Check your GitHub repository to confirm all files are uploaded.
